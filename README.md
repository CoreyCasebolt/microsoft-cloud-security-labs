# Microsoft Cloud Security Labs

This repository contains hands-on labs focused on Microsoft cloud security, identity administration, monitoring, and automation. These projects were built to strengthen practical skills with Azure, Microsoft Entra ID, Microsoft Defender, Microsoft Sentinel, Conditional Access, KQL, and Logic Apps.

## Projects

| Project | Focus |
|---|---|
| Entra ID Identity Security Lab | Users, groups, MFA, Conditional Access, audit logs |

## Skills Practiced

- Microsoft Entra ID administration
- Conditional Access and MFA
- Identity and access management
- Cloud security monitoring
- Audit and sign-in log analysis
